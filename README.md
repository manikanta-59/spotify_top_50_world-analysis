📖 Overview

This project analyzes the Spotify Top 50 World dataset to uncover trends in music popularity, artist performance, and audio features. The goal is to transform raw data into meaningful insights using data cleaning, exploratory analysis, and interactive dashboards.

🛠️ Tools Used
Microsoft Excel (Data Cleaning & Preprocessing)
Power BI (Dashboard & Visualization)


📊 Data Cleaning (Excel)
Removed duplicates and handled missing values
Standardized column formats for consistency
Cleaned and formatted artist names and track details
Prepared dataset for accurate analysis in Power BI
📈 Dashboard Features (Power BI)
🎵 Top Artists by Popularity
📊 Distribution of Song Features (Danceability, Energy, Tempo)
🌍 Popular Genres Analysis
🔥 Top Tracks Ranking
📅 Trend Analysis of Music Features
🎧 Comparison of Audio Characteristics


🔍 Key Analysis
Identified the most popular artists and tracks globally
Analyzed how audio features influence song popularity
Explored patterns in danceability, energy, and loudness
Compared genre-wise performance and trends


💡 Key Insights
High danceability and energy are common in top-ranking songs
Certain artists dominate the global charts consistently
Pop genre has the highest representation in Top 50
Songs with higher loudness and tempo tend to perform better


📌 Conclusion

This project demonstrates how data cleaning in Excel and visualization in Power BI can be combined to extract valuable insights from music data. It highlights trends that can help understand listener preferences and music industry patterns.
